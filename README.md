# FleetOps

FleetOps is a fleet management web application designed to simplify the management of vehicles, drivers, and daily fleet operations through a clean and user-friendly interface.

## Features

- User Authentication
- Dashboard
- Vehicle Management
- Driver Management
- Responsive User Interface

## Technology Stack

### Frontend
- React
- Vite
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open the application in your browser.
