# FleetOps

FleetOps is a fleet management web application designed to simplify the management of vehicles, drivers, and daily fleet operations through a clean and user-friendly interface.

## Features

- User Authentication
- Dashboard
- Vehicle Management
- Driver Management
- Responsive User Interface

## Repository Structure

This repository is structured as a MERN stack monorepo, divided into separate frontend and backend workspaces:

```text
FleetOps/
├── frontend/             # React + Vite + Tailwind CSS v4 Client
└── backend/              # Node.js + Express REST API
```

## Technology Stack

### Frontend
- React
- Vite
- HTML
- CSS (Tailwind CSS v4)
- JavaScript

### Backend
- Node.js
- Express.js

## Getting Started

### 1. Frontend Client
To run the React dashboard client:
```bash
cd frontend
npm install
npm run dev
```

### 2. Backend Server
To configure and run the Express server:
1. Create a `.env` file in the `backend/` directory based on `backend/.env.example`.
2. Run the development server:
```bash
cd backend
npm install
npm run dev
```
